# django-database-project
## 1. Cafe Management System – Feature List 
Menu Management: Maintain and display a complete list of food items and beverages.


Custom Drink Builder: Allow customers to create their own drinks with multiple customizable options.


Customer Order Cart: Add, update, or remove items before placing an order.


Billing & Payment Processing: Calculate total bill, accept cash payments, and generate change details.


Digital Receipt Generation: Send receipts to customers via SMS or email.


Sales Dashboard for Owner: View daily, weekly, and monthly sales reports.


Kitchen Order Tracking: Monitor order status and progress in the kitchen.


Inventory Management: Track stock levels, update quantities, and receive low-stock alerts.






## 2. PC Builder System – Feature List
Product Selection: Customers can browse and choose individual PC components.


Multiple Build Carts: Users can create and manage multiple carts for different PC builds.


Bottleneck Detection (API Integration): The system checks CPU–GPU compatibility and identifies potential bottlenecks.


Smart Part Recommendations: Suggest components based on bottleneck analysis and estimated power consumption needs.


Wishlist Management: Customers can save desired parts for later.


Build Sharing: Users can share their completed or ongoing builds with friends.


Pre-built PC Listings: Display a curated list of ready-to-purchase pre-built systems.


PSU Calculator: Estimate required power supply capacity based on part selection.


Product Reviews: Allow users to view and submit reviews for components.


User Accounts: Support login, logout, and basic account management.
